Yet another github account analyser.

Gives a simple github overview and a quicklink section for a github username entered.

## About

This project is a **GitHub Repo Viewer** built with React and Tailwind CSS. It fetches GitHub user data via the github api and displays them in a neat layout with necessary information.
<br>Built to learn React + Tailwind and api interactions.

---

## Features ##

- View basic info of a user like the the user name, social links, a few quicklins and their repositories.
- Clickable links to all the necessary sections
- A little hover interactivity

---

## Tech Stack

- **Frontend:** React, Tailwind CSS  
- **APIs:** GitHub REST API  
- **Icons:** FontAwesome

## Installation

1. **Clone the repo**

```bash
git clone https://github.com/Phoenix2066/yetAnotherGitAnalyser.git
```
2. **Install dependencies**

```bash
cd project-name
npm install
```

3. **Run the project**
```bash
npm run dev
```
The app should now be running at http://localhost:3000/

---

## Usage

1. Enter a GitHub username in the input field.
2. The app fetches all Github user data and gives a basic preview for the user data and repositories of the user along with a few quicklinks.
3. Click on the links to navigate to the required page.
